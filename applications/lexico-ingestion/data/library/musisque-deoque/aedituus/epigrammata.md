# epigrammata

**1** Dicere cum conor curam tibi, Pamphila, cordis,

**2** Quid mi abs te quaeram, uerba labris abeunt,

**3** Per pectus manat subito <subido> mihi sudor:

**4** Sic tacitus, subidus, <dum pudeo>, pereo.

**5** Quid faculam praefers, Phileros, qua est nil opus nobis?

**6** Ibimus sic, lucet pectore flamma satis.

**7** Istam nam potis est uis saeua extinguere uenti

**8** Aut imber caelo candidus praecipitans;

**9** At contra hunc ignem Veneris, nisi si Venus ipsa,

**10** Nulla est quae possit uis alia opprimere.
